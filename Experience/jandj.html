Mechanical R&D Intern

The first summer, I designed and prototyped a mechanism to allow the tip and tilt of a lens in an optomechanical system for an excimer laser. This system must meet the following requirements:

  • Allow precise, individual, adjustment of lens tip and tilt to align the laser
  • A robust locking mechanism to hold the desired position
  • Easily accessible for service teams
  • Cheaper to produce than off-the-shelf components

Additionally, I used both SolidWorks and COMSOL finite element analysis to conduct several design studies to guide system frame layout, structure, and material choice to meet requirements while accounting for input from service, industrial design, electrical teams, etc.

Lastly, I utilized PolyJet, SLA, and FDM printing technologies to produce prototypes, marketing samples, and fixtures for validation studies.

The following summer I redesigned and prototyped an existing 2 DOF retractable arm used for system calibration, reducing system complexity and halving the number of components used. I utilized linear bearings and a counter-force spring to achieve the same motion with equivalent accuracy, all within a smaller form factor.

Next, I got involved with the electrical engineering team to create a test bench graphical user interface (GUI) in C++ for a USB to serial communication platform. The GUI allows users to configure serial communication protocols (I2C and SPI) and send data to embedded electronics for testing and validation purposes. Additionally, the GUI allows users to control general purpose input/output pins as well as create/open scripts for more efficient testing.

Lastly, I investigated complaints from production and conducted root-cause analyses to correct errors from suppliers and revise production processes.
